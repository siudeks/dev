# siudeks.github.io

## test locally

```bash
hugo serve
```

## Upgrade

- Check [current Hugo version](https://github.com/gohugoio/hugo/releases)
- update version number in devcontainer.json
- rebuild dev container
- check version unsing
  ```bash
  hugo version
  ```
- test locally if is working well
  ```bash
  hugo server
  ```
